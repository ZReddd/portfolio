AP Price Compare

Project Overview

AP Price Compare is a full-stack web application that helps Australian consumers compare grocery prices across major supermarkets. The Vue.js frontend provides a responsive interface while the .NET 8 backend handles data processing and API services.

This ongoing project currently supports price comparison for thousands of products with weekly specials tracking.

Vue.js Frontend Implementation

The frontend is built with Vue.js using a component-based architecture. Key UI elements like the sidebar, product cards, and buttons are modular components composed in App.vue. Tailwind CSS provides utility-first styling for rapid development and consistent design across the application.

The main application interface showing price comparison and the Vue component code structure with Tailwind CSS classes.

.NET 8 Backend with Real-time Price API

The backend system is built with .NET 8 and fully managed through Swagger UI, featuring real-time supermarket price integration:

Dynamic API Endpoints:

•GET http://localhost:[port]/api/Coles/{product_id}/detail - Retrieves full product details

•GET http://localhost:[port]/api/Coles/{product_id}/price - Returns current pricing data

•Automatic price synchronization with supermarket databases

•Swagger-powered API documentation and testing interface

•Entity Framework Core for efficient data operations
